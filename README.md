📊 Challenge Data Science – Telecom X

📌 Sobre o Projeto

Este repositório contém o desenvolvimento do Challenge Telecom X, proposto pela Alura, cujo objetivo é analisar dados de clientes de uma empresa fictícia de telecomunicações, com foco na identificação de fatores associados à evasão de clientes (churn).

A partir de técnicas de análise exploratória de dados (EDA), o projeto busca gerar insights que auxiliem na tomada de decisões estratégicas voltadas à retenção de clientes.

🎯 Objetivo

* Analisar o comportamento dos clientes da Telecom X e identificar padrões relacionados ao churn, considerando variáveis como:

* Tipo de contrato

* Tempo de permanência (tenure)

* Valores cobrados

* Perfil de consumo e pagamento

🛠️ Tecnologias Utilizadas

* Python 3

* Jupyter Notebook

* Pandas

* NumPy

* Matplotlib

* Seaborn


🔍 Etapas da Análise

* Entendimento do problema

* Carregamento e inspeção dos dados

* Limpeza e tratamento dos dados

* Análise exploratória de dados (EDA)

* Geração de insights

* Conclusões e recomendações estratégicas

📊 Principais Insights

* Clientes com contrato mensal apresentam maior taxa de churn.

* O churn é mais frequente entre clientes com menor tempo de permanência.

* Clientes que cancelam os serviços possuem, em média, valores mensais mais elevados, indicando possível insatisfação com o custo-benefício.

📈 Conclusão

A análise demonstrou que o churn está fortemente associado a contratos de curto prazo, menor tempo de relacionamento com a empresa e valores mensais elevados. Esses resultados reforçam a importância de estratégias focadas na retenção de clientes nos primeiros meses de contrato e na oferta de planos mais atrativos.
